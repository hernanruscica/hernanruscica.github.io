Juego de Memoria "MemoTest"

Ver Código Live Demo
Volver

Descripción:

Simple juego de memoria, estilo "MemoTest", hecho con puro javascript, CSS, HTML y programación estructurada. Dinámicamente genera los elementos del DOM para hacer las cartas y ponerlas en el correcto lugar.
Simple animación con CSS para "dar vuelta" las cartas. Usa la función "setTimeOut" para contar el tiempo.

Características:

  • Puedes Jugar al juego de memoria.
  • Puedes Traducir la interfaz y los mensajes hacia el usuario.
  • Puedes Reiniciar el juego
  • El jugador es controlado por tiempo, además se cuenta su puntaje e intentos restantes

Habilidades:

  • Document Object Model
  • Html
  • Diseño responsivo
  • Animaciones CSS
  • Manejo del tiempo con setTimeOut
