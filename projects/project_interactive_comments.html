Sección de comentarios interactivos

Ver Código Live Demo
Volver

Descripción:

Una sección de comentarios interactivos hecho con puro javascript, CSS, y HTML. Al desafio original de "Frontend Mentor": Interactive comments section , yo le agregué la característica de almacenamiento en el local Storage, para mantener los datos de prueba, entre sesiones.

Características:

  • Puedes agregar un comentario, borrarlo o editarlo.
  • Puedes responder un comentario, borrarlo o editarlo
  • Puedes votar los comentarios o respuestas
  • Guardado del estado en el local storage del navegador

Habilidades:

  • Document Object Model
  • Html
  • Diseño responsivo
  • Nomenclatura BEN
  • Local Storage
