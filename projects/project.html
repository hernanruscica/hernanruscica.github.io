Descripción:

Sitio Web diseñado desde cero, pensando colores, tipografias e iconos acordes a la tematica. Separado en componentes reutilizables, utilizando Figma. Proyecto en FIGMA

El maquetado fue hecho con Bootstrap y CSS personalizado, utilizando HTML semántico.
La parte frontend de esta aplicación esta construída con Bootstrap y prototipo hecho con Figma.

Características:

  • Diseño responsivo (Mobile y PC escritorio)
  • Diseño UX con Figma
  • Archivos optimizados para una carga rápida en todos las velocidades de internet.
  • HTML semántico para mejorar posicionamiento del sitio

Habilidades:

  • HTML
  • CSS
  • BOOTSTRAP
  • JAVASCRIPT
  • Diseño responsivo
  • Nomenclatura BEN
